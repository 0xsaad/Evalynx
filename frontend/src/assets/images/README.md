# Images Folder

## Logo Instructions

Please add your logo image file to this folder with the name `logo.png` (or `logo.svg` for better quality).

### Requirements:
- **Format**: PNG or SVG (transparent background recommended)
- **Filename**: `logo.png` or `logo.svg`
- **Recommended size**: 
  - For PNG: 200x200 pixels or higher
  - For SVG: Any size (vector format)
- **Background**: Transparent/backgroundless for best results

### Current Usage:
- Navbar: Logo displayed next to "EvalynX" text
- Footer: Logo displayed as an icon

Once you add your logo file, the components will automatically use it.
